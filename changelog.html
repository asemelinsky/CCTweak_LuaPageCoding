Історія змін Lua Editor для ComputerCraft
v7 2026-04-11 Latest
  • ↔
    Resizable panels
    Перетягуй сплітери між редактором, консоллю та бічною панеллю. Налаштуй робочий простір під себе.
  • ⏩
    Inline console input
    Функція read() тепер показує поле вводу прямо в консолі замість браузерного вікна. Також sleep() реально чекає (до 10 сек).
  • ℹ
    Status bar
    Рядок стану внизу: позиція курсора (Ln/Col), кількість рядків, поточний сервер.
  • ⚡
    Покращений Lua-транспілер
    Підтримка конструкцій: while..do repeat..until not / and / or ~= #table for..in pairs/ipairs nil. Додано моки: string.* math.* textutils.* colors.
  • 🖥
    Peripheral API
    Моки периферійних пристроїв: peripheral.find() peripheral.wrap(). Підтримка Monitor (write, clear, setCursorPos, setTextColor...) та Modem (open, transmit, close...).
  • 📋
    Шаблони програм
    10 готових шаблонів: Tunnel, 3x3 Tunnel, Quarry, Staircase, Tree Farm, Redstone Toggle, Redstone Pulse, Monitor Output, Modem Send, Fuel Check, Inventory Dump.
  • 📑
    Покращені вкладки
    Drag-and-drop для зміни порядку вкладок. Подвійний клік на назві вкладки для перейменування. Ctrl+T — новий таб.
  • 🎨
    Dark / Light тема
    Перемикання теми кнопкою в sidebar. Вибір зберігається між сесіями.
  • 🌐
    Два ігрових сервери
    Вбудована підтримка двох серверів: Server 1 (Forge 1.20.1) та Server 2 (NeoForge 1.21.1). Обирай сервер у випадаючому списку — файли зберігаються на відповідний сервер. Також можна додавати власні SFTP-сервери.
  • 💻
    Локальний режим
    Режим «Локальний (мій ПК)» — зберігай файли прямо на свій комп'ютер без SFTP. Обери папку computercraft/computer/ і працюй офлайн.
