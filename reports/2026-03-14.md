### 2. Переписано PHP → Node.js API для Vercel

Vercel не підтримує PHP, тому backend переписано:

| Старий файл | Новий файл |
|-------------|------------|
| `upload.php` | `api/upload.js` |
| `list_files.php` | `api/list_files.js` |

Використовується npm пакет `ssh2-sftp-client` для SFTP з'єднання.

`index.html` оновлено — всі fetch виклики тепер ідуть на `/api/upload` і `/api/list_files`.

---

### 3. Додано нові файли

- **`api/upload.js`** — serverless функція: завантажує .lua файл на сервер через SFTP
- **`api/list_files.js`** — serverless функція: список файлів, читання, видалення
- **`package.json`** — залежність `ssh2-sftp-client`
- **`vercel.json`** — rewrites `.php` → `/api/*` (для зворотної сумісності)
- **`server.html`** — інфо-сторінка про новий сервер (характеристики, підключення, моди, проблеми міграції)
