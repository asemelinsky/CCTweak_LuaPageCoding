### 2. Локальний режим — File System Access API
**Коміт:** `461812e`

Додано опцію **"Локальний (мій ПК)"** у перемикач серверів. Дозволяє зберігати, читати та видаляти `.lua` файли прямо на локальному комп'ютері без SFTP-з'єднання.

- Кнопка "Обрати папку ComputerCraft" через File System Access API браузера
- Працює з папкою `computercraft/computer/{N}/`
- Не потребує сервера чи інтернету

**Статистика:** +166 / -20 рядків (index.html)

### 3. Локальний Python-сервер (незакомічений)
**Файл:** `local_server.py` (untracked)

Написано автономний Python HTTP-сервер як альтернативу для браузерів без File System Access API:
- Емулює `upload.php` та `list_files.php` через файлову систему
- Потрібно лише вказати `WORLD_PATH` у конфігурації
- Запуск: `python local_server.py` → `http://localhost:8080`
